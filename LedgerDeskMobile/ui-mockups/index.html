Design rationale

What changed from v2

Navy instead of black

Deep navy (#0A1628) as the base gives warmth without losing premium feel. Surfaces step up in blue tint instead of gray, following Material Design elevation.

MUI component patterns

Contained / outlined / text button variants, outlined TextField with floating label, FAB with Material shadow, Paper-style cards with soft dividers. Feels consistent with any modern MUI app.

Generous auth padding

Login and Activation now use 28px horizontal + 32px top padding. Buttons and inputs no longer touch screen edges. Form rhythm uses 16–20px gaps for breathing room.

Ornament removed

No italic emphasis words, no uppercase tracked-out labels, no gold underline accents, no editorial wordmarks. Labels are normal-case, muted color. Clean, not magazine-y.

Flat hero

Dashboard balance card is now a plain surface with a clean number — no radial glow or gradient. Lets the data lead. Income / expense cards use color only on the label.

Simpler typography

Headlines at weight 600 (not 800), letter-spacing −0.3px (not −1.2px). Money numbers get tabular figures at weight 700. OS font only, no web fonts loaded.
